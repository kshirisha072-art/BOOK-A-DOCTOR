# BOOK-A-DOCTOR

## 📖 Overview

**Book a Doctor** is a full-stack MERN-based healthcare appointment booking system that enables patients to search for doctors, book appointments, and manage their medical consultations online. Doctors can manage appointments and patient requests, while administrators oversee the entire platform. The application provides secure authentication, appointment management, online payment support, medical report management, and an intuitive user interface.

## ✨ Features

### 👤 Patient Module

* Patient Registration & Login
* Browse Doctors by Specialization
* Book Doctor Appointments
* View Appointment History
* Manage Profile
* Online Payment
* Download Medical Reports

### 👨‍⚕️ Doctor Module

* Doctor Registration & Login
* Manage Appointments
* View Patient Details
* Update Appointment Status
* Upload Medical Reports
* Doctor Dashboard

### 👨‍💼 Admin Module

* Admin Login
* Manage Patients
* Manage Doctors
* Manage Appointments
* Manage Medical Reports
* Dashboard Overview

### 🌍 Additional Features

* JWT Authentication

* Appointment Scheduling
* Medical Report Management
* Online Payment Integration
* Responsive User Interface
* MongoDB Atlas Database

## ⚙️ Installation

1.*Clone the Repository git clone https://github.com/kshirisha072-art/BOOK-A-DOCTOR* 

2.*Navigate to the Project cd BOOK-A-DOCTOR*

3.*Install Backend Dependencies cd backend npm install*

4.*Install Frontend Dependencies cd ../frontend npm install*

5.*Configure Environment Variables*

6.*Create .env files inside both backend and frontend folders and add the required environment variables.*

7.*Start Backend cd backend npm run dev

8.*Start Frontend cd frontend npm start
